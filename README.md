#  CityGuide - 城市文旅导览平台

这是我正在开发的城市文旅导览平台。
我希望做一个能真正帮游客快速找到景点、查看游记的系统。目前项目还在不断完善中，第一阶段已经跑通了核心流程。

---

## 目前已经实现的功能

### 1. 用户登录系统
- **手机号验证码登录**：写了正则校验，保证手机号格式正确。验证码存放在 Session 中，实现了基本的登录和注册逻辑。
- **自定义拦截器**：为了不让用户频繁登录，我写了两个拦截器，一个用来刷新 Token 状态，一个用来校验有没有访问权限。

### 2. 景点模块
- **分类展示**：支持按照不同的类型（比如自然风光、历史古迹）查看景点。
- **详情查询**：点击景点能看到详细信息。
- **Redis 缓存优化**：考虑到景点查得比较频繁，我用 Redis 把景点信息存起来了，这样就不用每次都去翻数据库，响应速度快了很多。
